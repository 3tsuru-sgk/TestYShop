リファラー検証ポータル

Shopifyのアクセスコントロール機能をテストするための起点ページです。

【アクセス不可】ドメインのみ送信で遷移 【アクセス許可】詳細リファラーを送信して遷移

検証のヒント:
・ドメイン間遷移の場合、通常リファラーはドメイン情報しか送信されません
・許可ページをドメインではなく、詳細URLにする場合は、遷移元でリファラーコントロールの設定が必須です。
・ブラウザのデベロッパーツール（F12）を開き、コンソールログで Final Verdict が ALLOW になるか確認してください。
